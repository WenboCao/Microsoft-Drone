This file contains documents on Kalman Filter and data compression. 

The matrix calculation part is from EJML, which is a collection of classes for matrix calculation. 

Compression_GPS.java is the file in which defines parameters of Kalman Filter. The file path is ejml-master/examples/src/org/ejml/example.

CSVReader.java is the file which used to receive UTM data from CSV file. Its path is ejml-master/examples/src/org/ejml/example.
